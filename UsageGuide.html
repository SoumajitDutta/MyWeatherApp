☛ Get current weather data by city, ZIP code, or geolocation coordinates (latitude/longitude).

Parameters

☛ lat (required) - latitude of desired location.

☛ lon (required) - longitude of desired location.

— or —

☛ zip (required) - 5-digit Zip code (United States only)

— or —

☛ city (required) - city name.

☛ state (optional) - US state (for United States cities only).

☛ country (optional) - country name.

Go to the search button(Top-right corner of the website) and search the city name for which you want to get the weather report :)

✓ API provides the latest weather information for any city or geographic location in the world.
